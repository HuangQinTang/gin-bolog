项目结构
```
gin-blog/
├── conf
├──  middleware
├── models
├── pkg
├── routers
└──  runtime 
```
- conf：用于存储配置文件
- middleware：应用中间件
- models：应用数据库模型
- pkg：第三方包
- routers 路由逻辑处理
- runtime 应用运行时数据
